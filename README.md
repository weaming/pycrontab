# PyCronTab

```
pip3 install pycrontab
```

```
$ pycrontab -h
usage: pycrontab [-h] [-c CONFIG] [-l {INFO,DEBUG,WARNING}]

optional arguments:
  -h, --help            show this help message and exit
  -c CONFIG, --config CONFIG
                        config file path
  -l {INFO,DEBUG,WARNING}, --log-level {INFO,DEBUG,WARNING}
                        logging level
```
